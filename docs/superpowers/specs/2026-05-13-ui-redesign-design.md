# UI 重设计规格 — 简伴

日期：2026-05-13

---

## 设计目标

将现有 React 单页应用的 UI 从深绿色暗色风格重设计为「温暖陪伴」风格，适配单用户个人工具场景，后续通过 Tauri 打包为桌面应用。

---

## 色彩系统

| Token | 值 | 用途 |
|-------|----|------|
| `--cream` | `#F5F0E8` | 全局背景 |
| `--cream-2` | `#EDE7DA` | 分割线、次级背景 |
| `--amber` | `#C47A3A` | 主色：激活态、进度、按钮 |
| `--amber-2` | `#A36128` | 深琥珀：悬停、强调 |
| `--amber-bg` | `#FBF3E8` | 琥珀浅底：卡片气泡背景 |
| `--text-1` | `#3A2E22` | 主文本 |
| `--text-2` | `#7A6A56` | 次要文本 |
| `--text-3` | `#B0A090` | 辅助文本、占位符 |
| `--white` | `#FFFFFF` | 卡片、侧边栏底色 |

## 布局结构

```
┌──────────┬─────────────────────────────────────┐
│  64px    │                                     │
│  icon    │         主内容区（滚动）               │
│  侧边栏   │                                     │
└──────────┴─────────────────────────────────────┘
```

### 侧边栏（64px，固定）

- 白底，右侧 1px `--cream-2` 分割线
- 顶部：Logo 图标（绿色渐变圆角方块）
- 中部导航图标：首页、计划、习惯、复盘、周报
- 底部：AI 监督、设置
- 激活态：`--amber-bg` 背景 + 左侧 3px `--amber` 竖条指示

### 主内容区（flex column，padding 32px 36px）

区块从上到下：
1. **问候头部** — 日期小字 + 大号问候语 + 右侧 streak 标签 + 头像
2. **概览卡片行**（3 列等宽）— 今日计划 / 习惯打卡 / AI 提示
3. **详细任务列表**（全宽白色卡片）
4. **数据统计条**（4 个 pill，横排）

---

## 各模块规格

### 今日计划卡片

- 左：72px SVG 环形进度图（`--amber` 描边，`--cream-2` 轨道，圆角端点）
- 右：任务名列表（已完成划线 + 灰色，未完成正常色）
- 头部右上角：`x/y` 完成计数 badge

### 习惯打卡卡片

- 每行：emoji 图标 + 名称 + 连续天数 + 5 点阵（最近 4 天 + 今天）
- 点颜色：已打卡 `--amber`，未打卡 `--cream-2`，今日用略大点强调

### AI 提示卡片

- 气泡样式（左下尖角），背景 `--amber-bg`，正文 `--text-2`
- 机器人 emoji 悬于气泡左上角外
- 底部两个按钮：「稍后提醒」（ghost）+ 「开始复盘」（主色填充）

### 详细任务列表

- 白色大卡片，圆角 14px
- 每行：圆角方形 checkbox + 任务名 + 颜色标签 + 时间
- 已完成：checkbox 填满 `--amber`，文字 `--text-3` + 删除线
- 末行：虚线 + 图标的「添加新任务」行
- 标签色系：工作（蓝）/ 健康（绿）/ 成长（紫）/ 习惯（灰）

### 数据统计条

- 4 个等宽 pill，白底圆角 10px，`--shadow-sm`
- 每个：emoji 图标 + 数值（大字加粗）+ 标签（小字灰）

## 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 26px | 600 | 问候语 |
| Label | 13px | 600 | 卡片标题、导航 |
| Body | 13–14px | 400 | 任务名、正文 |
| Caption | 11–12px | 400–500 | 时间、标签、辅助 |

---

## 交互说明

- **Checkbox 点击**：立即切换完成状态，调用 `PATCH /plans/{id}` 或 `PATCH /habits/{date}`
- **「添加新任务」**：展开内联输入框，回车提交
- **侧边栏导航**：切换页面，无刷新（React 状态切换）
- **「开始复盘」按钮**：跳转到复盘页面
- **「稍后提醒」**：暂无后端，前端 toast 提示「已记录，18:30 再提醒」
